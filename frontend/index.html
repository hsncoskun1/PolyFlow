ℹ️ API Anahtarlarını Nasıl Alırım?
1. polymarket.com'a giriş yapın
2. Sağ üst → Profil → Settings → API bölümüne gidin
3. "Create API Key" butonuna tıklayın
4. API Key, Secret ve Passphrase bilgilerini bir yere not alın (yalnızca bir kez gösterilir)
5. Private Key için: MetaMask → Hesap → Export Private Key
🏦 EVM Özel Anahtar ZORUNLU
Polymarket hesabınıza bağlı Ethereum cüzdanının özel anahtarı. İşlem imzalamak için kullanılır. MetaMask → Hesap → Export Private Key
🔐 API Anahtarı (API Key) ZORUNLU
Polymarket CLOB API erişim anahtarı. pk_live_... formatında başlar.
🛡️ Secret ZORUNLU
API oluştururken verilen gizli anahtar. Yalnızca bir kez gösterilir.
🗝️ Passphrase ZORUNLU
API imzalama parolası. Oluştururken belirlediğiniz veya atanan parola.
💳 Funder Adresi OPSİYONEL
İşlemleri finanse eden EVM cüzdanının genel adresi (0x...).
🔗 Relayer Ayarları
Relayer, Polymarket'in gasless claim/redeem servisidir. Kazanan pozisyonların USDC'sini gas fee ödemeden tahsil etmek için gereklidir. polymarket.com → Settings → API bölümünden alabilirsiniz. Trade (alım/satım) için gerekli değildir.
Otomatik Tahsilat özelliği bu alanlar doldurulmadan aktif edilemez.
🔑 Relayer API Key OTO CLAIM/REDEEM İÇİN ZORUNLU
Polymarket relayer servisi API anahtarı. UUID formatında.
📍 Relayer Adresi OTO CLAIM/REDEEM İÇİN ZORUNLU
Polymarket relayer cüzdan adresi (0x...).
